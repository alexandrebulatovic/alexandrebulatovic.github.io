### Réalisation d'un site e-Commerce PHP en binôme

*Les fichiers du site sont en ligne :* https://github.com/alexandrebulatovic/projet_ecommerce

Le but du projet était de créer un site e-commerce vendant des produits quelconque (dans notre cas des smartphones) utilisant la technologie PHP et réutilisant ce que l'on avait appris en cours de programmation web.

**Compétences :**
- utilisation des langages HTML, CSS et PHP
- utilisation d'une base de données MySQL pour la persistence des données (utilisateurs, produits et commandes)
- utilisation de formulaires avec transmission GET, POST
- utilisation des variables COOKIE et SESSION
- sécurisation du site avec l'affichage de pages d'erreur lorsque l'utilisateur "bidouille" l'URL
- création des requêtes SQL nécessaire au fonctionnement du site
- création d'un schéma relationnel de base de données adapté aux informations que l'on veut stocker
- utilisation d'une architecture MVC

# Lors de la licence professionnelle ACPI

### Réalisation d'une application android en binôme

Le but du projet était de réaliser une application Android qui "parse" (analyse) un fichier XML mis à disposition par le professeur sur internet et dont l'URL est à notre disposition. Des URL d'images et des informations sur ces images sont contenues dans le fichier XML. L'application stocke ces images dans sa mémoire cache et les affiche à l'utilisateur dans un menu. L'application devait être jolie (design) et facile à utiliser. En termes de fonctionnalités, il fallait pouvoir sélectionner une image et zoomer avec les doigts.

En collaboration avec [Abdel Ait Abdelloili](https://fr.linkedin.com/in/ait-abdelloili)

**Compétences :**
- utilisation du langage JAVA avec Android Studio
- "parsing" de fichier XML issu d'une URL web pour récupérer des informations
- stockage d'images dans la mémoire cache du téléphone
- utilisation d'un menu pour naviguer entre les différentes images et permettre la sélection
- création d'un "écran" (view) permettant de zoomer sur une photo avec les doigts
- utilisation des composants [Material Design de Google](https://fr.wikipedia.org/wiki/Material_design) en gardant la rétrocompatibilité avec les appareils qui ne supportent pas ces nouveaux composants
